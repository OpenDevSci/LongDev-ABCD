Overview

This example assesses whether height, measured at the 1-Year follow-up in a subsample of ABCD participants, is predicted by weight and height measured at baseline. A visual inspection is further conducted via a scatterplot to graphically represent the relationship between height at Year_1 and weight at baseline, controlling for height at baseline. The ensuing analysis and interpretations are detailed in the subsequent sections.

Results

Compute Multiple Regression Model

The code snippet below tells R to conduct a multiple regression analysis by subtracting each participant’s height at T1 from their height at T0. Relevant summary statistics are also provided.

# Reshape data from long to wide format
df_wide <- df_long %>%
    pivot_wider(
        names_from = event,
        values_from = height,
        names_prefix = "Height_"
    )

# Fit the regression model excluding NA values
model <- lm(Height_Year_1 ~ Height_Baseline + handedness, data = df_wide, na.action = na.exclude)

# Calculate summary statistics
model_summary <- summary(model)
print(model_summary)
